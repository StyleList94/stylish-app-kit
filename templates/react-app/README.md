# Stylish React App

Quick start your React Application

## Featured

- Vite React Typescript
- SWC
- Vitest
- Testing Library
- ESLint Airbnb rules
- Prettier

## Get Started

### Use template

```bash
pnpm create stylish-app -t react
```

### Install Dependencies

```bash
npm install

# or pnpm
pnpm install
```

### Run App

```bash
npm run dev

# or pnpm
pnpm dev
```

### Testing

#### CI/watch

```bash
npm run test

# or pnpm
pnpm test
```

#### UI Mode

```bash
npm run test:ui

# or pnpm
pnpm test:ui
```

#### Coverage

```bash
npm run test:coverage

# or pnpm
pnpm test:coverage
```
